# Terminal Chess Engine ♟️

A terminal-based two-player chess engine written in **Python**.  
The program simulates a playable chess game with rule-based move validation and check detection.

## Features
- 8×8 **matrix-based board representation**
- **Turn-based gameplay**
- **Legal move validation** for all pieces
- **Piece capture mechanics**
- **Check detection**
- **Castling support**
- Unicode **chess piece visualization** in terminal
